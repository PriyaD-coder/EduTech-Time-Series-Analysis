# Task 14 --- Time Series Analysis using ARIMA

## Project Overview

This project demonstrates Time Series Analysis and Forecasting using the
AirPassengers dataset. The goal is to analyze temporal patterns and
predict future values using ARIMA.

## Objectives

-   Time series visualization
-   Decomposition (Trend, Seasonality, Residuals)
-   Stationarity testing (ADF Test)
-   ARIMA model building
-   Forecasting future passenger values

## Dataset

Monthly airline passengers from 1949--1960.

## Tools Used

Python, Pandas, Matplotlib, Statsmodels, Jupyter Notebook

## Workflow

1.  Load and visualize dataset
2.  Decompose time series
3.  Perform ADF test
4.  Apply differencing to make data stationary
5.  Build ARIMA(1,1,1) model
6.  Forecast next 24 months

## How to Run

pip install statsmodels Open notebook and run all cells.
